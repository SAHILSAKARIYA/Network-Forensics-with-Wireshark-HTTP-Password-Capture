# Network Forensics with Wireshark – HTTP Password Capture

## Overview
This project demonstrates basic network traffic analysis using Wireshark. The lab focused on capturing HTTP login traffic and analyzing insecure credential transmission over unencrypted protocols.

## Objectives
- Capture HTTP traffic using Wireshark
- Analyze HTTP POST requests
- Identify plaintext credential transmission
- Understand the importance of HTTPS encryption

## Tasks Performed
- Captured HTTP traffic from a demo login page
- Filtered packets using Wireshark
- Analyzed HTTP POST request data
- Identified usernames and passwords transmitted in plaintext
- Documented security risks associated with HTTP

## Tools & Technologies
- Wireshark
- HTTP Protocol

## Skills Learned
- Network Traffic Analysis
- Packet Inspection
- HTTP Protocol Analysis
- Credential Security Awareness
- Basic Network Forensics

This project was performed in a controlled lab environment for educational purposes only.
